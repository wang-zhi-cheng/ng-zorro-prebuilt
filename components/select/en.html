Select

Select component to select value from options.

When To Use#

  • A dropdown menu for displaying choices - an elegant alternative to the native<select>element.
  • UtilizingRadiois recommended when there are fewer total options (less than 5).

Examples

Basic Usage.

Search the options while expanded.

Select with tags, transform input to tag (scroll the menu)

Coordinating the selection of provinces and cities is a common use case and demonstrates how selection can be coordinated.

Using theCascadercomponent is strongly recommended instead as it is more flexible and capable.

The value ofngModelcomes from thenzValueofnz-option, when thenzValueofnz-optionis an object, thengModelwill be an object too, the usage ofcompareWithis the same asSelectControlValueAccessor.

A complete multiple select sample with remote search, debounce fetch, ajax callback order flow, and loading state.

The height of the input field for the select defaults to 32px. IfnzSizeis set to large, the height will be 40px, and if set to small, 24px.

Multiple selection, selecting from existing items, you can max selected 3 option at the same time.

Usingnz-option-groupto group the options.

Custom the content of nz-option vianzCustomContent.

Load data on scroll.

API#

<nz-select>
  <nz-option nzValue="lucy" nzLabel="Lucy"></nz-option>
</nz-select>
