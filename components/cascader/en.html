Cascader

Cascade selection box.

When To Use#

  • When you need to select from a set of associated data set. Such as province/city/district, company level, things classification.
  • When selecting from a large data set, with multi-stage classification separated for easy selection.
  • Chooses cascade items in one float layer for better user experience.

Examples

Cascade selection box for selecting province/city/district.

Separate trigger button and result.

Disable option by specifying thedisabledproperty inoptions.

Cascade selection box of different sizes.

Reset control value by Ng Reactive Form.

Allow select option only onnzChangeOnreturn true.

Search and select an option directly.

Specifies default value by an array, butnzOptionsisnull/empty, data are loaded bynzLoadDatafunction.

Specifies default value by an array.

Hover to expand sub menu, click to select option.

Allow only select parent options.

For instance, add an external link after the selected value.

Load options lazily withnzLoadData.

Show Cascade selection box in a modal dialog.

Show menu on mouse enter and hide on mouse leave.

Specifies default value by an array, and settingnzOptionsin a asynchronous way.

<nz-cascader [nzOptions]="options" [(ngModel)]="values"></nz-cascader>
