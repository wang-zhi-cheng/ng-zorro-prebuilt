Tree

When To Use#

Almost anything can be represented in a tree structure. Examples include directories, organization hierarchies, biological classifications, countries, etc. TheTreecomponent is a way of representing the hierarchical relationship between these things. You can also expand, collapse, and select a treeNode within aTree.

Examples

The most basic usage, tell you how to use checkable, selectable, disabled, defaultExpandKeys, and etc.

Drag treeNode to insert after the other treeNode or insert into the other parent TreeNode.

To load data asynchronously when click to expand a treeNode.

Tree With Line.

Customize directory tree(with nzTreeTemplate, supported contextmenu).

basic controlled example.

nzBeforeDrop: Validate before drop event, for example: only allow to drag inside(delay 1s)、 validate using modal .etc.

Searchable Tree.

You can customize icons for different nodes.
